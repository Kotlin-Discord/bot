KDBot

KotDis Bot

This repository contains what will eventually be the primary bot used for managing the Kotlin Discord server.

Right now, it's in a very early state.

Running with Docker

In order to run the bot with Docker and Docker Compose, you'll need to run a local copy of our website.

Getting the Containers Ready

In the site project root, you'll need to run

npm i
./gradlew build

In the bot project root, you'll need to run

./gradlew build

You'll also need to create an .env file in the bot project root and set the BOT_TOKEN variable to your test bot token.

BOT_TOKEN=...

Additionally, the docker-compose file assumes that both the site and bot root folders have been placed within the same directory. You can override this by setting the SITE_PROJECT_PATH environment variable to the path to the site project folder.

Running the Bot and Site

Simply run docker-compose up in the bot project root to launch both containers.
